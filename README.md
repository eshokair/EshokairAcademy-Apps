## 🏛️ Project Architecture &nbsp;|&nbsp; هيكل المشروع

```
Eshokair Apps System
│
├── 🗄️  Database Layer
│     ├── Schema Design & Security
│     ├── Lookups & Master Data Tables
│     └── PL/SQL Business Logic
│
├── 🔐  Security Layer
│     ├── Authentication (Login Flow)
│     ├── Authorization (Roles & Permissions)
│     └── Global Session Items
│
├── 🌐  Application Layer (Oracle APEX)
│     ├── Page Types & Navigation
│     ├── Dynamic URLs & Deep Linking
│     └── Global Items & Shared Variables
│
└── 🎨  UI Layer
      ├── Header (Parts 1 & 2)
      ├── Footer
      └── Body Layout & Components
```
