# E. Shokair Academy - Eshokair Apps Project

هذا المستودع هو المرجع البرمجي والتعليمي المتكامل لمشروع **Eshokair Apps**، والذي يهدف إلى بناء منظومة تطبيقات ويب احترافية باستخدام **Oracle APEX** بتقنية "أقل كود" (Low-Code)، مع التركيز على منطق الأعمال (Business Logic).
